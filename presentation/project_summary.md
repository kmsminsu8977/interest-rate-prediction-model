# 발표 초안

## 1. 문제 정의

물가, 고용, 기간스프레드, 신용스프레드는 금리 상방/하방 압력을 어떻게 다르게 만든다?

## 2. 데이터와 가정

- 합성 샘플 데이터: `data/sample/macro_rate_panel.csv`
- 원본 강의자료/코드 직접 사용 없음
- 재현 가능한 baseline 실행을 우선 구성

## 3. 방법

거시 변수의 방향성 신호를 단순 점수화해 다음 금리 변화 압력을 추정한다.

## 4. 현재 산출물

- 실행 스크립트: `python -m src.run_baseline`
- 결과 표: `outputs/tables/baseline_results.csv`

## 5. 후속 작업

- 실제 데이터 연결
- 민감도 분석
- 차트/보고서 산출
- 프로젝트별 상세 검증
